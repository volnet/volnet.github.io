## 推荐序 德鲁克给我的十条教谕（吉姆·柯林斯）

吉姆·柯林斯，用自己的话，总结了十条教谕，清晰全面地概况的全书精要，每一小段都十分精辟，后续的段落，德鲁克将用它的智慧娓娓道来，期待开启后续的每一页。

## 前言

> 有效性是管理者获得报酬的原因，无论他们是需要对自己和他人绩效负责的经理（manager），还是只需要对自身绩效负责的专业的个体贡献者（individual professional contributor）。他们付出的聪明才智再多，花费的时间再多，如果有效性为零，“绩效”就为零。

1. 有效性（effectiveness），至关重要，它是管理者所必须的！
2. 卓有成效的管理者，不是天生的，只能通过后天习得，而且不难，遵守**为数不多的几种惯常做法**就可以获得这样的有效性。
3. 把这些惯常的做法变成习惯，就能够变得有效。

## 引言 有效管理者是由什么造就的

在作者熟知的一些优秀人士身上总结了8个惯常习惯（practice）：

- 他们会问“什么事是必须做的？”
- 他们会问“什么事对企业来说是正确的？”
- 他们制订行动计划
- 他们承担起决策的责任
- 他们承担起沟通的责任
- 他们专注于机会而不是问题
- 他们召开高效的会议
- 他们用“我们”而不是“我”来思考和说话

## 获得必需的知识

有效的管理者做事的原则：

1. 做必须做的优先级最高的事。
2. 做对企业（而不是股东、股价、员工或者管理者等）来说正确的事。

### 制订行动计划

> 行动计划是对意图的陈述，而不是死板的要求。不能让它成为束缚，而是应当经常修改，因为每一次行动，无论成败，都会带来新的机会。

行动计划是希望我们能够把事儿想清楚，但是计划里总是不能预估所有的变化，因此当变化出现的时候，我们应该积极进行调整。

### 行动

这一节用四个关键议题来描述：决策、沟通、机会（而非问题）和会议。

1. 决策：关键事项的负责人、截止时间、影响、相关人员。让有能力做决策的人去做这个决策，并定期进行评估决策的质量。决策不一定有效，但不决策注定无效。
2. 沟通：需要把行动计划和决策传达出去，同时要想办法把信息同步回来。得为这个结果去不懈努力。
3. 机会：只有新的机会会创造成果，解决问题最多只能降低损失。
4. 会议：两个及以上的人沟通其实就是开会，因此会议并非严谨的会议室会议。因此这里说的是有效利用时间。必须目的明确、事前准备、高效推进、跟进落实，总之，你不能漫无目的地浪费自己和他人的时间。

### 用“我们”来思考和说话

我们，是指把自己和所在组织视为一个整体，以组织的利益为优先，因为管理者之所以获得某种职权，是因为组织的利益需要他去这么做。

有效可以学会，也必须学会。

## 第1章 有效是可以学会的

> 才智、想象力和知识都是至关重要的资源，但只有有效性才能把它们转化为成果。至于这些资源本身，只是为能够取得什么样的成果设下了界限。

这句话一下把**有效性**和其他价值区分开来。

### 为何需要有效管理者

知识工作者区别于体力工作者，前者以思考为工作，而别人看不见摸不着，后者则有大量的评价工具产出数量、质量等来衡量，评价后者的方法对前者不适用。

> 因此，知识工作者必须做一件体力劳动者不需要做的事——提供有效性。他可不能指望自己的产出直接产生效用，就像一双做工精良的鞋子那样。

**提供有效性**，从原文的这个关键词看，知识工作者应该让自己“被看见”，这也是衡量他价值的重要方法。

### 谁是管理者

需要作出决策知识工作者，都是管理者。它不因职位高低和大小而定。

### 管理者面对的现实

管理者想要有效的关键就是它们必须主动拥抱有效性，克服这些管理者必然面临的困境：

1. 时间被人占用：即便你很努力，可能你不得不接听上司或客户的电话，被占用很多与有效性无关的时间。
2. 忙于日常事务：应该做真正重要的事情，而不是按照事情发生的顺序去做事情。
3. 管理者更应该关注平行部门：知识工作者的有效性是由知识成果被他人使用来界定的，而使用这些成果的人是在外部，比如你的平行部门，而不是你的下属。
4. 组织内部无成果可言。成果全部存在于组织外部：因为只有客户/消费者可以为企业创造价值，它们在企业外部，且与企业无关。而管理者所接收的外部信息通常是有限的，管理者必须要主动想办法去关注外部，获取更多外部的信息。

### 有效带来的希望

虽然雇佣更优秀的人是个不错的建议，但是人类的大量历史证明，普通人还是占大多数，因此提高他们之间的协作效率是至关重要的。

现在的专业分工，每一个细分领域都可以深不见底，将不同的细分领域整合在一起则需要相互之间更加有效，我们不能寄希望于找到全才，因为它的供给太少了，但是我们可以寄希望于有效性，提高这些资源（专业的知识工作者，人，专才，而不是全才）的有效性将有助于提高整个组织的有效性。

### 有效真的可以学会吗

有效，作为一项本领，不依赖于天赋，而是习惯。它需要经过简单的一些实践，反复练习，并真正形成习惯。

而这些习惯，在后续的章节中逐一解释。

## 第2章 认识你的时间

资源决定完成工作的上限，时间是最稀缺的资源，且供给弹性为零，无法储存。有效管理时间，不是关注于工作计划，那只是美好的愿望，关注时间花在哪才能解决如何高效管理时间的问题。

### 管理者面对的时间需求

拥有大块的整段的时间才有可能取得有效性，而管理者的时间通常容易被碎片化，这个和管理者总是被各种琐事骚扰有关。要想要提高时间的有效性，可行的方法就是“记录时间被用在了哪里”。

### 时间诊断

把自己实际使用的时间，如实地实时的记录下来，而不是事后回忆。然后你会发现大多数情况下，时间的分配和你预想以及期望的分配情况有所不同。

接下来是系统性地管理时间：

1. 砍掉那些根本不需要做的，即做与不做毫无差别的事情。
2. 把那些可以让别人做的事情标记出来，也许别人做得比你更好。
3. 减少那些自己作为管理者而浪费别人时间的行为，一个简单的方法就是坦诚地问对方：我做的哪些事，对你的有效性没有贡献，所以是浪费你时间的？

### 精简浪费时间的活动

除了上面3种浪费时间的行为，还要防止管理和组织不善造成的时间浪费：

1. 缺少体系或者远见而浪费时间的活动：应该把经常出现的问题标准化，变成标准动作，而不是每次都需要一惊一乍。
2. 人员过剩导致的时间浪费：人多就容易产生内耗，那些为完成工作偶尔才需要的职能应该放在团队外面。
3. 组织结构缺陷导致的时间浪费：典型特征就是会议太多，每个人不能独立行事。会议在本质上是对组织结构有缺陷所做的补救。开会就不能工作，二者不可兼得。
4. 信息功能失灵导致的时间浪费：信息没有有效传递，不同人对相同的信息需要额外的时间去处理后才能得以利用，这都会造成时间的浪费。

### 集中“可自由支配的时间”

把时间集中起来利用，将可以大大提高有效性。

> 我至今没有碰到过什么危机，是不能等上90分钟再去处理的。

时间管理的最后一环就是把经过记录和分析之后发现的、通常属于管理者可以自由支配的时间集中起来。

时间是最稀缺的资源，不管好它，也就管不好任何其他事情。此外，时间分析是对工作加以分析，并且明确其真正重心的唯一既简单易行但又系统的方法。

## 第3章 我能做出什么贡献

> 一个眼里只有工作本身、看重下向权力的人，无论职位有多高，也只是一个“下属”。相反，一个重视贡献、对成果负责的人，无论职位有多低，都是名副其实的“高管”。他让自己对整个组织的绩效承担责任。

有效的管理者，更关注自己应该为组织承担什么责任，而不是工作本身。

### 管理者自己的追求

> 思考“我能做出什么贡献？”这个问题，就是探寻这份工作当中尚未得到发掘的潜力。就许多职位而言，如果把能做的贡献充分发掘出来，过去取得的那些绩效纵然辉煌，相比之下也会黯然失色。

有效管理者通常会问自己“我能做出什么贡献”，将它作为自己的工作目标。他将有可能变得卓有成效。

### 如何让专才有效

> 这意味着专才必须弄清楚谁会使用自己的产出，以及使用者必须知晓和理解哪些东西才能让自己生产的这些片段发挥成效。

让专才有效的办法就是他把自己的贡献和别人的工作结合在一起。他有义务让别人理解他的工作成果。

### 正确的人际关系

良好的人际关系：他们在处理自己的工作以及人际关系时都**专注于贡献**。这导致他们建立的关系富有成效。

相处融洽和言语和悦不仅没有意义，而且不过是恶劣态度的虚假门脸罢了。

**专注于贡献**本身可以满足有效人际关系必须具备的四个基本要求：

- 相互沟通：沟通不是上级要求下级，这样通常无效，而是让下级肩负责任。
- 团队合作：一般的专业人士在组织里从属于不同的专业部门，但团队里则需要他们和很多不同专业的人相互协作。
- 自我发展：对贡献的专注，才会带来自我发展。
- 培养他人：专注于贡献的管理者会让团队里的其他人也变得一样专注于贡献。

### 有效的会议

有效的会议，要有明确的目的，专注于解决具体的问题，而不应该是漫无目的的。

> 专注于贡献可以让管理者解决自己面临的一个基本难题：各种事务杂乱如麻，光看事情本身无法判断哪些重要，哪些纯粹是“噪声”。以贡献为标准就能做出判别。专注于贡献为管理者提供了组织原则，提供了判断事项相关性的标准。
>
> 专注于贡献还让管理者把工作情境中的一个先天劣势转变为优势之源：管理者需要依靠他人，且身处组织内部。专注于贡献可以让他把众多个体塑造成团队。
