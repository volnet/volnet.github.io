## 前言

1993年夏，美国《外交》季刊发表了我的一篇文章《文明的冲突？》这篇文章中提出的论点：正在出现的全球政治主要和最危险的方面将是不同文明集团之间的冲突，引起了人们的各种反应，包括新奇、义愤、恐惧和困惑。

这本书是这篇文章的延伸，目的在于提出一个对于学者有意义的和对于决策者有用的看待全球政治的框架或范式。
