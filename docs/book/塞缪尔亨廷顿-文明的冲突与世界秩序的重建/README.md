# 文明的冲突与世界秩序的重建

THE CLASH OF CIVILIZATIONS AND THE REMAKING OF WORLD ORDER

[美]塞缪尔·亨廷顿 著

## 中文版序言

这本书是美国哈佛大学教授塞缪尔·P·亨廷顿于1993年夏季由在《外交季刊》上发表了一篇题为[《文明的冲突？》（英语：The Clash of Civilizations）](https://en.wikipedia.org/wiki/Clash_of_Civilizations)文章中的观点。

## 前言

1993年夏，美国《外交》季刊发表了我的一篇文章《文明的冲突？》这篇文章中提出的论点：正在出现的全球政治主要和最危险的方面将是不同文明集团之间的冲突，引起了人们的各种反应，包括新奇、义愤、恐惧和困惑。

这本书是这篇文章的延伸，目的在于提出一个对于学者有意义的和对于决策者有用的看待全球政治的框架或范式。

## 第一部分 一个多文明的世界

### 第一章 世界政治的新时代

> 这一章讲述了本书将以“七八个文明”来划分世界，进而解释整个世界政治。

冷战时期的世界被分为两极三部分，美国阵营（富）、苏联阵营（略穷）、其他（穷）。

20世纪80年代后，随着共产主义世界的崩溃，冷战的国际体系成为历史。对国家最重要的分类不再是冷战中的三个集团，而是世界上的七八个主要文明。美国、欧洲、中国、日本、俄罗斯、印度、伊斯兰国家。未来的冲突将由文化因素而不是经济或意识形态所引起。而最危险的文化冲突是沿着文明的断层线发生的那些冲突。

冷战是杜鲁门提出的对于美苏格局的一个简化的范式。在冷战结束后，也出现了类似一些必要的范式，它们都或多或少地从不同的视角解读了冷战后的新的世界秩序：

1. 一个世界，欢欣而和谐。但事实上，虽然冷战结束了，但是各方势力并没有就此进入全面和平，各方势力慢慢形成了新的冲突和意识。
2. 两个世界：我们和他们。人们喜欢用二元论来区分我们的和你们的。前者包括西方和日本，它拥有大约15％的世界人口，后者包括其他所有的人。
    - 以富裕程度进行划分：前者富有后者贫穷，但冲突通常是在富有国家之间或贫穷国家之间的冲突，而富有国家和贫穷国家之间，则只有在前者想要殖民后者的前提下才会产生，但随着各个解放战争的胜利和殖民者自身放弃殖民，相对而言富余国家和贫穷国家之间的冲突其实是越来越少的。——所以基于富余程度划分，难以解释各个阵营之间的冲突。
    - 以文化差异进行划分：把世界分成西方的和非西方的。这种划分方式没有太大的意义，因为非西方的国家之间，事实上在宗教、社会结构、体制和普遍价值观方面是各不相同的。
3. 以国家为边界，将世界分成184个国家。这种划分有其一定的道理，主要是因为在国际政治中国家是常见的单元，国家和国家之间处于无政府状态，依靠实力来平衡各自的关系。但这样的划分的问题是：所有的国家都以同样的方式看待自己的利益，并以同样的方式行动。它简单地假定实力是理解国家行为的唯一出发点，但却没有达到深入的理解。
4. 完全的混乱。它以世界所发生的冲突为主脉络来描绘世界。它虽然构造于现实，但是世界并不是完全无序，也并非由冲突构成的，它对于理解世界秩序可能帮助不大。
5. 从七八个文明来看待世界（本书所提倡的看待世界的范式）。它不像一个世界和两个世界的范式那样，为了简化而牺牲现实；但它也不像国家主义和混乱范式那样为了现实而牺牲简化。它提供了一个易于把握和易于理解的框架，可以用来理解世界，在越来越多的冲突中区分不重要的和重要的，预测未来的发展，为政策制定者提供指导路线。它也建立在其他范式的要素的基础之上，并结合了这些要素。它与其他范式之间较之那些范式彼此之间更加相容。
    - 世界中的整合力量是真实的，而且正在产生对文化伸张和文明意识的抵消力量。
    - 世界在某种意义上是一分为二的，主要的区分存在于迄今占统治地位的西方文明和其他文明之间，然而，其他文明之间几乎没有任何共同之处。简言之，世界是划分为一个统一的西方和一个由许多部分组成的非西方。
    - 民族国家是而且仍将是世界事务中最重要的因素，但它们的利益、联合和冲突日益受到文化和文明因素的影响。
    - 世界确实是无政府主义的，充满了部落和民族冲突，但是给稳定带来最大危险的是那些来自不同文明的国家和集团之间的冲突。

### 第二章 历史上的文明和今天的文明

历史上很多学者提出了对于文明的观察和解释：

1. 从单一文明过渡到多元文明：最初提出文明的观点是由18世纪法国思想家相对于“野蛮状态”提出的。把欧洲文明作为单一文明的标准，用这个标准来衡量其他社会的文明化程度。然而事实上世界并不是真正意义上的单一文明，不同文明之间有各自不同的文明的标准，因此多元文明才是更合理的描述文明的方式。因此，多元文明是更加包容和开放的文明观念，更符合当今世界的多元化趋势。
2. 文明和文化本质上是一致的：在德国之外，文明被看作是一个文化实体。——这一段是否可以这样理解：在德国思想家眼里，文明和文化是两个不同的东西，而在世界的其他地方，文明则是被看作是一个文化的实体。从作者的观点来看，文明和文化都涉及一个民族全面的生活方式，文明是放大了的文化。不同文明之间的差异，来自于不同群体之间价值观、信仰、体制和社会结构之间的差异，它是多种因素促成的结果。
3. 文明是各个相似文化的人群的聚集。文明之间是不同社会群体文化各个维度的最大公约数，也就是一个文明之间在各个维度的相似度都显著高于与另一个文明在这些维度的相似度。但是维度非常多，因此，文明之间并没有严格的界限，即便是人们也曾经试图去重新定义和划定边界，但是文明之间的界限总是并不特别清晰。但是从宏观的角度讲，不同文明之间确实又有着相对更大的差异。文明是对人最高的文化归类，是人们文化认同的最广范围，人类以此与其他物种相区别。文明是具有包容性的，它超越于乡村、地区、种族群体、民族、宗教群体之上，将不同的人聚集起来。也就是说文明和文明之间是有很明显的互异性的，但即便如此，文明也并非有清晰的边界，文明也随着时间的变化在不同的文明之间也发生着吞并和分裂。
4. 文明并不是一成不变的，而是不断演化的。它通常有混合、孕育、扩张、冲突时代、普遍帝国、衰败和被入侵七个阶段。
5. 文明是文化的实体，但文明本身不履行政治职能，因此一个文明可能包含一个政治实体，可能包含多个，大多数文明都包含两个以上的政治实体。
6. 人们对文明的划分方式各不相同标准不一。但是至少这12个文明大家都认可：其中7个文明已不复存在（美索不达米亚文明、埃及文明、克里特文明、古典文明、拜占庭文明、中美洲文明、安第斯文明），5个仍然存在（中国文明、日本文明、印度文明、伊斯兰文明和西方文明）。除这5个文明之外，或许还应加上东正教文明、拉丁美洲文明，可能还有非洲文明。

有了以上的讨论，下面引出对每个文明的说明：

1. 中国文明，又称为中华（Sinic）文明，儒教文明，包括中国和中国以外的东南亚以及其他地方华人群体的共同文化，还有越南和朝鲜的相关文化。
2. 日本文明，既有被认为是和中国文明相同的文明，也被视为独立的文明，是中国文明的后代。
3. 印度文明，在南亚次大陆存在着一个或一个以上的相继的文明。这些文明一般被称为Indian、Indic或Hindu文明，人们更喜欢用后者来称呼最近的那个文明。
4. 伊斯兰文明，伊斯兰教起源于公元7世纪的阿拉伯半岛，然后迅速传播，跨越北非和伊比利亚半岛，并向东伸展到中亚、南亚次大陆和东南亚。
5. 东正教文明，以俄罗斯为中心的东正教文明，它区别于西方基督教文明，是因为它源自拜占庭文明，不同于西方的宗教；那些地区遭受过鞑靼人200年统治，以后又是实行君主专制制度，受文艺复兴、宗教改革、启蒙运动等西方重大事件的影响是有限的。
6. 西方文明，学者们一般认为它有三大组成部分，分布于欧洲、北美和拉丁美洲。西方就包括欧洲、北美，加上其他欧洲人居住的国家，如澳大利亚和新西兰。
7. 拉丁美洲文明，拉丁美洲文明是欧洲文明的后代，但它却是沿着非常不同于欧洲和北美的道路演进的。它具有社团主义的、独裁主义的文化，而这种文化在欧洲的程度要小得多，在北美则根本不存在。欧洲和北美都感受到宗教改革的影响，并且把天主教和新教文化结合在一起。从历史上看，虽然可能会有所变化，但拉丁美洲一直仅仅是天主教的世界。拉丁美洲文明结合了一些本土文化，这些文化不曾存在于欧洲，在北美也已被有效地消灭，而在以墨西哥、中美、秘鲁和玻利维亚为一方和以阿根廷和智利为一方两者之间有重大的不同。拉丁美洲的政治演变和经济发展与流行于北大西洋国家的模式大相径庭。
8. 非洲文明，这是一个存在争议的文明，因为非洲文明被欧洲文明的影响太深了。

宗教可能是划分不同文明的重要因素，世界四大宗教基督教、伊斯兰教、印度教和儒教至少起到了这样的作用，而第五个宗教佛教，因为在历史传播的过程中杯融入了当地文化，因此这些文化实体并没有共同的佛教宗教认同，所以它虽然是个重要的宗教，但它无法用于区分不同的文明。

人类历史的文明，最初是在各个不同的地区独自发展起来的，相互之间没有太多的交流。然后随着各自思想和技术的发展，文明和文明之间慢慢有了微弱的联系，相互之间也偶有发生征服。欧洲的基督教一开始并不占优势，后来吸收了伊斯兰文明和拜占庭文明的思想，又在文艺复兴之后，整个欧洲产生了显著的技术进步，航海技术和军事进步，让欧洲文明开始向外扩张，并产生覆盖了地球上大多数土地，现在的整个国际体系都是建立在西方文明的基础上的。但是这并不代表仅有一个文明，而是一个文明在这段时期里成为了主导。西方文明产生了自由主义、社会主义、无政府主义、社团主义、马克思主义、共产主义、社会民主、保守主义、国家主义、法西斯主义和基督教民主，并在世界各地进行演化，但非西方社会远不只是西方创造的历史的客体，而是日益成为它们自己的历史和西方的历史的推动者和塑造者。所以世界文明由西方文明为主的单一文明占主导作用，演化为多极文明。先是走上了美苏其他三极，然后走向了前面提到了七八个文明的多极文明。

### 第三章 普世文明？现代化与西方化

一些观点认为普世文明正在崛起，也就是世界趋同。但事实并非如此，各个文明仍然蓬勃发展，以下是一些认为普世文明在扩大势力的观点：

1. 认为世界人民有共同的价值观。但事实上，即便是不同的文明，相互之间也有很多相似之处，这些古已有之，根本不是新的趋势。
2. 城市和识字带来了文明的趋同。但事实上，这些会带来文明的传播，但是并不因此就导致文明单一化。
3. 达沃斯经济论坛代表了各个文明的趋同力量。但事实上，这些只是各自文明中的少数代表，先不说这些人在其文明体内部是否有代表性，他们聚在一起更多是为了全球经济活动、贸易等行为。
4. 西方消费品，如麦当劳、可口可乐和好莱坞电影的传播。人们可以接受西方的商品，但并不会导致对自身文明的放弃，接受商品不等于改变文明。而文化的传播稍显复杂，但是娱乐也不等于改变文明。至少从事实上，这些商品和文化已经传播多时，但是各自的文明正在以各自的形式在加强，而不是被同化。
5. 语言，有人认为英语的普及导致普世文明。首先英语确实是世界语言，但是更多时候它是不同文明之间交流的工具，而不是因为使用了这样的语言就放弃了自身的文明。在世界各地，即便是那些曾经被殖民并强制使用英语的地区，也在逐步恢复自身的语言，这恰恰说明了，语言更多时候只是工具，不会改变人们对于自身文明的认同。
6. 宗教，一直都有，而且始终有着自己的人口基数，这些基数在这些年里有变化但是相对稳定，很难有一家独大的宗教，因此宗教并不会导致普世文明。

普世文明的观念通常来自下面三个论证：

1. 随着美苏冷战结束，因为苏联解体了，所以世界会变成美国自由主义模式，但事实上并非如此，世界反而走向了多极，而且运转的很好。
2. 世界经济贸易的发展，导致相互之间的贸易、投资、旅游、媒体和电子通信的增长带来了世界趋同。但事实上，趋势向好的时候，世界是和平的，但是趋势变差的时候，世界则充满了文明之间的冲突。人们总是项目之间寻找差异点来识别“我是谁”。
3. 一个更常见的论点是认为现代化等同于西方化。虽然由西方引领的工业化、城市化等技术进步带来了工业革命、信息革命等，确实导致全世界都在发展相应的内容，但西方之所以是西方并不是因为这些新带来的技术变革所产生的，相反，它在这些技术变革之前，西方就已经是西方了，这些特征包括：古典遗产、天主教与新教、欧洲语言、精神权威和世俗权威的分离、法治、社会多元主义、代议机构、个人主义，这些特征的合体则是西方文明所独有的。所以可以说西方带来了现代化，但不能说西方化等同于现代化。

而其他文明对待现代化和捆绑而来的西方化，则有不同的态度：

1. 拒绝：在日本和中国早期的闭关锁国时期，这种思想是明显的。但是随着西方世界现代化，带来了战争和武力强迫这些想要隔离于世的难度越来越高。
2. 同时接纳西方化和现代化：以基马尔主义（凯末尔主义）所倡导的，其他文明对于现代化是不利的，因此要同时接纳二者。但是以其所建立的“新土耳其”为例，这件事的难度远超想象，基马尔使土耳其成了一个“无所适从”的国家，一个在其宗教、遗产、习俗和体制方面是伊斯兰的社会，但是其统治精英却决心使它成为现代的、西方的和与西方一致的。
3. （改良派）只接受现代化，不接受西方化：在中国的晚清时期，人们提出了一个口号：“中学为体，西学为用”。在日本，口号是“日本的精神，西方的技术”。这种想法将现代化的新技术引入，但仍然维持自己的文明。

在整个世界的演化中可以看出，改良派，也就是充分接收现代化，不仅没有使得自身的文明消失，反而增强了这种文明，并且相对减弱了西方文明。在伊斯兰文明中，虽然有一些条款可能会阻碍现代化，比如在诸如利息、斋戒、遗产法和妇女参加工作等方面可能与现代化发展存在冲突，但是在实现现代化的其他方面，文明本身并没有什么太大的冲突，而且，人们也在变通地接纳现代化。

总之，现代化并不一定意味着西方化。非西方社会在没有放弃它们自己的文化和全盘采用西方价值观、体制和实践的前提下，能够实现并已经实现了现代化。西方化确实几乎是不可能的，因为无论非西方文化对现代化造成了什么障碍，与它们对西方化造成的障碍相比都相形见绌。

## 第二部分 变动中的各文明力量对比

### 第四章 西方的衰落：力量、文化和本土化

西方实力依然最强但正在相对减弱，而这一切正是先发优势的作用正在减弱，具体而言是指，虽然推动生产力的“科学技术”发明自西方，但是却通过不断发展的信息网络、分工全球化等全球网络被非西方学习，进而提升非西方的相对实力，而且他们因为跟随西方的脚步，发展更加迅速，省去了不少试错的环节。

各种文明在西方崛起之前通常对自身文明非常自信，后来随着西方实力的增强，他们的领导人也通过西方化而获得了统治，在那个阶段各个文明都在效仿西方文明，认为西方文明是其实力增强的必不可少的一部分。但随着西方相对实力的减弱，新一代领导人在本土化的文化上成长起来，他们开始鼓吹自身传统文化、宗教的价值观，带来了各个文明的复兴。

现代化的进程带来了大规模城市化，新进入城市的这些无根基者感情和社会的需求无法被满足，但宗教恰恰填补了这样的空白，因此带来了宗教的复兴。宗教之间也是有互相替代的关系，那些能满足人们感情和社会需求的宗教替代了那些传统的但是无法满足这些功能的宗教，正如在韩国，基督教替代了佛教而成为了大量民众的选择。

### 第五章 经济、人口和挑战者文明

亚洲文化的复兴，是来自于的亚洲经济实力增长后，亚洲特别是东亚对自身文化的认同。他们认为经济增长的动力来自于自身文化的内核，包括日本、新加坡、中国等，他们认为自身文化中的“集体主义”对于经济实力的增长起到了至关重要的作用，认为西方应该向东方学习以补足自身文化上的缺陷。

> 一位评论家评述说，“天安门事件后的政权热切地接受把中国的民族主义作为合法性的新源泉”，并有意识地激起反美情绪来为自己的权力和行为辩护。一种中国的文化民族主义就这样产生了，用一位香港领导人1994年的话说就是：“我们中国人有了前所未有的民族感。我们是中国人并为此感到自豪。”在中国，20世纪90年代初出现了一种“回复到属于真正的中国东西的普遍愿望，这些东西常常是家长式的、排外的和专制的。在这种历史的重现中，民主就像列宁主义一样不被信任，因为它不过是另一个外国人强加的东西”。

伊斯兰教的文化复兴和亚洲不同，他们通过石油换取的财富让他们变得更加自信，同时随着城市化、信息技术的普及、先进技术的传播，结合根深蒂固的伊斯兰文化，二者的结合让其他文化无法渗入。而由于人口的急剧增长，同时相对于周边其他区域的人口停滞，穆斯林人口的迁移带来了伊斯兰教文化的传播，也给迁入地带来了文化上的影响。伊斯兰教也正在加强现代化的进程，让它变得与时俱进。

> 复兴运动将在社会内外留下一个伊斯兰主义的社会、文化、经济和政治组织构成的网络。复兴运动也将表明伊斯兰教是对道德、认同、意义和信仰问题的“解决方法”，但并不是对社会公正、政治压制、经济落后和军事弱小问题的解决方法。

从上面这句话，以及其他未摘录的内容。其实可以看出作者的观点。即便现在是多种文化并行，但是大家都认同改善经济和军事更多来自西方的先进技术，但大家发现发展这些并不需要太多的“西方文化”作为基础，甚至一些“亚洲文化”认为自身所强调的“集体主义”在赶超经济上更具优势。在这样的认知下，随着先进技术的普及带来的经济增长，文化自信也跟着生长，强者就容易被神话，又希望将自己的成功归因到自身的“文化基因”中。
