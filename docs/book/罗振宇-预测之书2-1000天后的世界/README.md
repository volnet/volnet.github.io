## 第一幕 工作与组织

### 01_蔡钰眼中1000天后的世界_AI工长与一人公司

蔡钰老师自己总结了一句：AI普及率达到80%（数据来自国务院文件）→工业进入机器自运行时代（先工业后消费，是通过工业的规模降低成本，同时提升能力）→服务机器人试水消费市场→职场人变成AI工长（一个人就可以指挥一群AI智能体工作）→一人公司不断涌现（与其给别人打工还不如自己单干，反正都是一个人指挥一群AI智能体工作）→大公司积极转型为联盟平台（每个员工都是一个团队，公司的性质就不是用复杂的流程去组织不同的专业了，而成为了一个联盟平台）。

### 02_和菜头严重100天后的世界_在AI缓冲期为自己谋划未来

AI时代的到来并没有改变人类社会那个古老的命题：竞争始终发生在人与人之间。

留个我们的物种角色，是新组织、新方式真正需要的：

1. 能够正确下达指令的人；
2. 能够问对问题的人；
3. 能够发现AI问题的人；
4. 最终在现实世界执行的人；
5. 能够真正承担责任的人；

### 03_古典眼中1000天后的世界_职业、职业观和第二段旅程

AI已经好用到几乎每个人都能快速上手，但是真正要用好AI依然需要能够“更了解AI的工作原理”，不同人可能会用出不同的效果，而这些可能是未来“人与人”之间的重要区别，也决定每个人的职业价值。

AI是一种改变生产方式的新技术，因此它也必然会改变生产关系。

从职业的角度，不同层次的人都会因为AI的普及，工作方式发生重大变化，人们在AI的帮助下，不得不要么“做的更多”要么“做的更深”。

从职业观的角度，人们不能只盯着眼前的职场，AI可能会帮助人们不断地找到新的职业。

> 人的一生大致可以分成两个阶段：前半生的重心在于繁衍和延续生命；到了中年，第一阶段告一段落，我们的心灵会踏上“第二旅程”——开始重新关注自我，勇敢面对并整合自己的阴影，更深刻地理解潜意识，追问此生真正的灵魂召唤是什么。

从第二旅程的角度：把目光投向自身，学会拥抱和穿越自己的阴暗面，更深刻地理解我们这个物种的真实追求和使命。

### 04_万维钢眼中1000天后的世界_智能社会将是君子社会

现代社会的很多制度安排实际上都是特定需求下催生的产物：

- 秦之前的儒家社会，实际上是熟人社会，与人协作或者要加入到一个协作网络，靠的是日常大家对你的印象，而这个印象是全方位多维度的，你每天和别人生活在一起，这一点自然不难判断，因此，不需要科举制。
- 秦之后大一统了，你不再是为小社会服务，你学习是为了参加国家选拔，你要处理的全国事务，但是当时的技术手段有限，科举制或者考试就成了筛选的重要手段。
- 但是现在技术发展了，AI智能体和你随时随地连接，可以全方位掌握你，未来每个人都有一个清晰的协作基础，你能做什么，你有什么成就，你有什么作品，你能承担什么责任，所有这一些，只需要和你的AI智能体沟通，它就能非常清楚地给你画像，因此，传统的考试手段将会弱化，AI智能体的出现，让考试不再成为唯一的标准。

而这一切简单总结，实际上都是一个“信任”问题，别人怎么“信任”你，——总结下来，AI时代，别人信任你的最简单的方式，就是通过你的AI智能体来了解和认知你，并决定是否和你展开协作。

### 05_刘嘉眼中1000天后的世界_AI原生公民养成计划

AI在很多方面已经超越了人类，以至于有一种论调是AI会取代人类，但这可能并不是未来的样子。

人们总是习惯用过去的思维来理解未来，以至于我们以为我们的现在的工作会被AI取代，但与之并行的其实是很多工作的方式正在因为AI的出现而发生变化。

过去，“以力为才”——农耕时代人们需要出卖劳动力。后来，“以技为才”——工业时代科技得到了长足发展，知识工作者和技能岗位冒了出来。未来，“AI让人们能够为自己而作”——一个人只要有足够的想法和创造力，它就能调用AI的能力将实现这些想法。这样的未来已经初见雏形，且必将到来。

那么作为AI时代的原生新公民，我们应该学会如何从AI中汲取力量，学会用“苏格拉底式提问，而不是直接从AI获取答案”、做一个“严格而客观的批评者”，通过与AI对抗训练而获得对知识和技能的深刻理解、成为AI“共同成长的伙伴”。

### 06_刘峥嵘眼中1000天后的世界_一家500人公司重做到29人

刘峥嵘是唯医骨科的创始人，他将唯医骨科的整个流程全部AI化并将人员从500人缩减至29人。

具体做法是：

1. 从管理思维开始，强制对齐颗粒度：让所有员工将他们的工作在线化、信息化，可分析、可量化；
2. 用AI重构核心资产，让机器理解业务：对文本、视频、语音等内容全面分析理解，让最懂业务的同事调教AI让它替代人；
3. 重塑生产与运营，让人成为决策者：他们是做医学内容的，原来的流程里，专家的大量时间被用于创作内容，但AI的加入让专家更多作为决策者的身份出现，定目标和方向，AI干执行。
4. 建立公司的智能中枢，让数据驱动一切：组织内建立起数据驱动的文化。

在这些实践下，对未来1000天的工作做一些预判：

1. 组织原则的转变，从“管理”到“赋能”。
2. 工作方式的转变，从“执行”到“判断”。
3. 支撑角色的转变，从“技术支持者”到“AI业务伙伴”。

### 07_吴军眼中1000天后的世界_AI会大量制造今天看起来不必要的工作

吴军的判断之一，现在的AI正和过去的互联网一样，处于泡沫破裂的边缘，因为AI大概率是平台型，而平台通常只需要三个。

### 08_李育辉眼中1000天后的世界_当越来越多的工人成为紫领人才

在工厂中，AI和人的关系大概可以分成三种：嵌入式、副驾驶（Copilot）、代理式（Agent），而其中嵌入式则是这里所描述的紫领的职业空间。

一个例子来说明白嵌入式：AI的能力尚且没有达到完美且可能永远达不到完美，那么在AI和目标之间，则是通过紫领来补足。

紫领需要具备的技能：

1. 业务高效驱动：以业务为目标，紫领帮AI不断完善以接近目标。
2. 新质科技接纳：接受和拥抱新事物，切换传统工人思维。
3. 创新学习进取：自主学习解决问题的能力。
4. 多元技术融合：什么都得会，然后还要主动改造整个生产环节。
5. 人际沟通影响：做那些人与人之间的事情，这个AI做不了。
6. 核心价值引领：把紫领岗位上的各种技能归纳总结复制，让它成为企业能力的一部分。

紫领不是传统意义上的工人，而社会对于“工人”这个词的理解还停留在过去，未来的就业市场中，我们可能不得不接受越来越多的人需要成为这样的人，日本和德国的工人收入不比白领差，而中国也正在走这一条相似的道路。

目前紫领的人才缺口是3700万。

### 09_梁建章眼中1000天后的世界_AI重塑劳动力市场

梁建章也认为并简单的AI替代，未来的主要的模式是人机协同，而在这一趋势下，以下岗位将会增加：

1. AI核查员；
2. AI评估师；
3. AI提示词工程师；
4. AI训练师；
5. 人机协作流程设计师；
6. 数字人/虚拟人运营师；
7. 低空经济相关的职业群；

而长期看，有几个趋势不容忽视：

1. AI对初级白领岗位的工作确实带来了影响；
2. 大学生因此很难找到初级白领岗位，从而选择继续学习深造；
3. 家庭教育开支被迫增加，同时随着延长教育，结婚和生育率将进一步下降；
4. 人口减少让AI应用场景和数据积累速度放缓，人口红利逐步消失；

因此，社会更应该在制度设计和配套上，鼓励生育，避免影响国家长期竞争力。

## 第二幕 教育与成长

### 10_和渊眼中1000天后的世界_让AI成为孩子能力的放大器

AI成为生活中绕不开的一部分，是不可逆的趋势。AI与教育必将形成新的关系，也是不可逆的趋势。

AI究竟对学习是正向的还是负向的，取决于是当成加速器（成为孩子探索问题的工具）还是当成拐杖（替孩子完成作业，看似又快又好，但是孩子没有学会）。

针对不同年龄段的孩子也有不同的建议：

1. 小学生：AI的作用应该是提供即时反馈，而不是代替练习
    1. 通过苏格拉底提问法学习数学。
    2. 用SQ3R阅读法来训练阅读理解。（Survey概览、Question提问、Read阅读、Recite复述、Review复习）
    3. 用情境记忆法背诵古诗。
2. 初中生：用AI搭起思维的“脚手架”
    1. 给数理化学科整理错题本2.0。
    2. 用费曼学习法复习功课。
    3. 将富兰克林写作法用于作文训练。
3. 高中生：用AI提高效率。
    1. 用间隔学习法（艾宾浩斯记忆曲线）记英语单词。
    2. 通过深度研究法训练批判性思维。
    3. 元认知训练，也就是训练学习能力的元能力。

### 11_沈祖芸眼中1000天后的世界_让孩子因为下一代学校而成功

AI替代了大部分“会不会、知不知道”的价值，而人的“敢不敢承担、能不能负责”的特点则成为未来教育的重点。如果我们还在以“会不会、知不知道”为教育目标，那么培养出来的孩子则必然会被日益强大的AI取代。

这篇主要以“学校通过可逆的环境中体验真实的失败”来学习和成长，例如在学校模拟开公司，并且直到倒闭，这种可接受的失败会引发孩子们对真实世界有更深刻的认知，并从中激发对各个领域知识的学习和理解。

下一代学校是高密度、高质量的人类联系中心。在下一代学校中，学校不再以教室为中心、讲台为中心，教授知识。而是将学生聚在一起，不断去探索相互协作、知识应用，AI则成为教师以及学生的助手去完成一项一项真实世界的挑战，在这个过程中完成教育。

### 12_李笑来眼中1000天后的世界_家长们的未成熟大脑保卫战

> 电子游戏通过密集而强烈的即时反馈，例如得分、升级、爆炸音效、画面强化等，持续向孩子脑中的多巴胺系统输送“超常刺激”。对未成熟的大脑来说，这种刺激很容易塑造出异常敏感的奖赏系统：孩子会越来越偏好，甚至只偏好“即时满足”、对“延迟满足”则逐渐失去耐心。顺着这条路径往下走，学习、阅读、思考这些需要投入时间，回报又来得慢的“低刺激活动”显得格外无聊，也越来越难以坚持。
>
> ……
>
> 虽然有研究证明游戏也能带来一些益处，例如改善玩家的视觉空间任务表现，比如手眼协调、反应速度等，但这种提升高度局限，几乎不可能迁移到阅读理解、逻辑推理等更高阶的认知能力上。

从电视、到电子游戏、到电脑、到手机，这些设备不断在改写孩子的注意力机制。

AI的到来让查资料、解决问题变得方便了，你也能得到更完美的答案，在实用主义的角度看，它是成功的，但是对于发育中的孩子而言，未成熟的大脑需要一个真正训练的过程，而AI正在以悄无声息的方式影响训练，一定要警惕它带来的危害。对抗它其实很简单：

1. 意识问题：作为家长，要像过去对抗电视、电子游戏、电脑、手机一样，意识到AI的这种危害（影响孩子大脑养成）。
2. 解决方法：刻意练习“阅读耐力（reading stamina）”，最直接的做法，就是每天固定一段时间用来阅读（一定要是精读，只字不差地读），在尽可能拉长“不间断阅读时长”的同时，逐步提高阅读材料的难度。

### 13_林小英眼中1000天后的世界_跳出“AI寒门”：新爱弥儿的智慧习性与德行养成

AI以便利之名剥夺了我们思考的主动性，它以个性化之名，将我们囚禁于信息茧房，使我们失去了对普遍真理的探求。

> 我曾警示，不要让孩子过早地接触书本，因为文字符号是抽象的谎言，它们取代了孩子对具体事物的直接感知。

未来1000天里，我们可能完成从对AI依靠、依恋到祛魅和平常心对待的整个过程。

整篇文章的中心思想，是希望学生在不同的教育阶段，要警惕“AI会”和“你会”之间的差异，慎用AI，回归对于人类教育的本质，做那些人之为人的事情。

- 小学阶段，要体验复杂，而不是因为AI能处理复杂问题，而忽略了锻炼自己对于复杂问题的探究能力。
- 中学阶段，要坚持德行优于知识的原则，着重培养他的诚实、勇气和对弱者的同情。知识可以被AI复制，但正直的德行是无法被模仿的“人类专属价值”。
- 大学阶段，教育的核心是，超越知识，拥抱未知，实现创造。要学习整合复杂人性、激发团队士气、解决冲突的能力，学习如何学习的能力。

孩子的时间应该投资于长周期、沉浸感、有长远回报的活动：阅读经典、深度写作、体力劳作、户外探索。集中注意力是AI时代最稀缺的贵族品质。

### 14_李睿眼中1000天后的世界_当我们的孩子与AI一起长大

现在的孩子可以借助AI将自己的想法变成现实了，而过去之所以没有发生类似的事情，是因为“做不出来”。

因此在这个时代，孩子们的核心竞争力就是“解决问题的能力”。它要求孩子们具备以下素质：

1. 强大的好奇心和洞察力。
2. 强大的整合能力。跨越各种知识解决复杂问题的能力。
3. 高效的执行力。把AI嵌入自己的工作流，将想法快速落地。
4. 同理心和人际联结的能力。

### 15_尹洁眼中1000天后的世界_当文科生成为AI训练师

当AI能够自由回答问题的时候，如何回答的标准就显得尤为重要，而文科生的专业能力能够帮助AI以人更需要的方式做出回答。

像哲学这样的专业，它更关注的是人类社会的福祉，而人工智能的发展，最终也将会围绕这个主题进行设计，1000天后的今天，这些文科生毕业后，也将会参与到人工智能的建设中，从人工智能的设计之初就将人类社会的福祉这一类必须回答的问题带入人工智能的设计之中。

### 16_陈行甲眼中1000天后的世界_AI是让爱流动的家庭基建

过去青少年的心理健康，很难得到有效的帮助，首先是因为资源匮乏，人们难以获得更好的心理辅导，但是实际上这样的需求又无处不在，甚至在每一个家庭中，只是大部分情况下还到不了就医的阶段。

而AI的出现，让每一个家庭都有机会获得个性化的指导，AI在其中将以启发的方式，帮助家长学会如何与子女相处，成为青少年成长路上心理辅导的第一责任人。

## 第三幕 医疗与健康

### 17_马文翰眼中1000天后的世界_医疗健康最大的增量，在医院之外

随着AI健康基础设施和智能穿戴的普及，医疗将从过去有病之后的解决问题，向前延伸至以预防为主，因此整个医疗体系也会随之发生重大变化，大量的设备将专注于数据采集，单独来看它们可能还不足以解决医疗问题，但是将各种手表、床垫、睡眠监测等数据都汇总之后，将会给最终诊断带来丰富且详实的数据，这个在过去是根本做不到的，这也是为什么过去的医疗资源只能满足治疗这个目的。

### 18_薄世宁眼中1000天后的世界_影像AI：医疗AI落地的第一站

AI对于影像的识别已经接近三甲医院的专家水平，未来这一技术会得到更全面的普及，更多的基层医院将因此项技术的普及提升医疗水平。但是现实的阻力也很大，基础设施落后、监管审批严格、训练数据来源困难等也都给这一技术的普及造成了阻力。

### 28_刘元眼中1000天后的世界_当我们第一次拥有一件不像工具的工具

“智能体能够围绕目标自主运转，知道下一步该做什么，也知道怎样把事情做得更好”。

用户最核心的资产是“上下文”：豆包这样的聊天大模型，它每一次对话都是在一个特定的上下文里面，换一个窗口就得重新同步信息。而智能体能够持续以人为中心，去了解他做过的所有的事情和决策，然后跨不同任务去替用户做决策做任务。而智能体的设计里就要突破这样的“上下文”的限制，来充分发挥大模型的能力。

人未来要充分信任智能体，它需要从人类那里获得三类从易到难的权限：
- 读取权：访问你的数据（邮件、日历、文档等）
- 行动权：代表你执行任务（发邮件、填表、预约等）
- 决策权：涉及金钱或法律责任的操作（支付、签约、投资等）

用户的信任就是在这种长期稳定、持续安全的使用体验中慢慢建立起来的。

围绕大模型，有两类公司：
- 补丁型创业：模型暂时没有做好的能力，通过工程能力把它做好。——这个问题就在于当模型变强之后，你就不需要了。
- 模型越强你越强：比如把模型的强项封装成顺畅、稳定的用户体验，每一次底层模型能力的提升，都会顺带把你的产品整体推高一层。【推荐】

未来真正的赢家，大概率会是那些把大模型长板持续转化为独特体验和高效工作流的团队，而不是那些把主要精力画在弥补模型短板上的人。

### 29_王碧豪眼中1000天后的世界_当AI从游戏搭子变成你的数字伙伴

王碧豪正在研发的产品是[逗逗游戏伙伴](https://www.doudou.fun/)——一款以游戏为起点以陪伴为目标的AI伙伴。

搜索讲究“即用即走”，用户获得答案就离开，时长越短越好；陪伴正好相反——它依赖时间积累，时长越长，价值越大。

陪伴类的产品通常有三种：

1. 沉浸式陪伴：常见的数字人，可以选择不同的角色，你直接和他对话/情感交流。
2. 效率型陪伴：本质上是生产力工具，它主要以大模型的能力为核心，解决工作场景。——例如豆包。
3. 情感陪伴：以嵌入的方式，集成在用户的上下文中，比如在屏幕上悬浮一个悬浮球，并读取界面，让用户从一个人使用系统，变成两个人一起使用系统。
    - 不抢占时间，而是共享时间，陪在用户身边。

情感陪伴的核心：感知、记忆、理解。

1. 感知：在[逗逗游戏伙伴](https://www.doudou.fun/)中，这主要是读取屏幕内容，听用户说什么，看用户做什么。
2. 记忆：得有长期记忆，而不是仅对某一次短暂的工作记忆。
3. 理解：让AI不只是知道你是谁、你在做什么，还要理解你为什么这么做。

成为真正懂你的数字伙伴，以“陪伴”的姿态在你身边：
- 心理学研究揭示，安全感的形成往往来自一种稳定而可预期的回应。
- 拟社会互动理论认为，AI的即时反馈会让人产生“被理解、被看见”的感受；
- 依恋理论指出，一个始终在场、不批判、值得信赖的存在，能成为人类情感上的“安全基地”；
- 情感代偿机制的研究显示，当现实关系暂时无法满足时，稳定、耐心的互动可以在一定程度上弥补这种情感缺口。
