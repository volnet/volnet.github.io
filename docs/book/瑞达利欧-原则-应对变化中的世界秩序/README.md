## 引言

太阳下没有新鲜事，很多事在你的一生里是第一次发生，但是在历史上却发生过很多次。通过研究历史，找寻历史事件的因果关系，将有助于你理解正在发生的各种事件，甚至可以帮你预测未来。

历史总是在两个周期中交替迭代：和平与繁荣时期 与 萧条、革命与战争时期。

研究的过程关注三大因素：

1. 长期债务与资本市场周期。
2. 内部秩序和混乱周期。
3. 外部秩序和混乱周期。

## 第一部分 世界是如何运转的

### 1 大周期简述

大周期总体上分为三个阶段：上升阶段→顶部阶段→下跌阶段，总是伴随着制度确立、财富累积、社会繁荣、分配不均、土崩瓦解等必然过程。

随着时间的推移，多代人的心理周期也在发生相似的变化（P60-P64，原文很精彩，强烈建议阅读），不同的时期也特别考验其领导人的理念：

1. 人民和他们的国家都很贫困，他们也认为自己很贫困。——早期新兴国家。因为穷，所以生活拮据、教育稀缺，为了转到生存以外可以用于投资的钱，通常要节衣缩食一代人，才能过度到下一阶段。
2. 人民和他们的国家都富裕了，但他们仍然认为自己贫困。——后期新兴国家。人们慢慢变富、有了储蓄、投资人力、基础设施、研发等，经济高速增长，值得投资。
3. 人民和他们的国家都富有，他们也认为自己富有。——健康高峰国家。1950-1965年，美国处于这个阶段。中国正在进入这一阶段。过去的基础设施等继续贡献价值，随着人们财富的累积，消费的偏好发生了变化，追求个性化、艺术、科技等蓬勃发展，劳动力越来越贵，新一代“没有受过苦”的人口逐渐占据了大多数，群体思想也慢慢发生转变，人们习惯性高消费。如果是大国，则成为世界重要经济体和军事强国，以获得其他国家的廉价劳动力和贸易优势地位。
4. 人民和他们的国家变贫困了，但他们仍然认为自己富有。——早期衰落国家。——债务相对收入上升，但是人们不愿意限制消费，入不敷出，不断增加债务来满足消费欲，国家还会因此增加战争开支以保护其全球利益。
5. 人民和他们的国家都贫困，他们也认为自己贫困。——明显衰落国家。央行和政府降低实际利率，提高名义GDP增长率，使其充分高于名义利率，从而减轻债务负担。

### 5 内部秩序和混乱大周期

内部秩序分为六个阶段：

1. 新秩序开始，新领导层巩固权力
2. 资源配置体系与政府官僚机构建立和完善
3. 出现和平与繁荣
4. 支出和债务严重过度，贫富差距和政治分歧扩大
5. 财政状况糟糕，冲突激烈
6. 出现内战/革命

以上六个阶段在一个国家内部随着时间循环出现。

国与国之间因为所处的阶段不同，各国在又在各个时期进行国际竞争，因此，从全球GDP的视角看，很多股力量是相互抵消的，导致全球趋势显得比较平稳。

### 6 外部秩序和混乱大周期

外部秩序和内部秩序很像，但是很多维持内部秩序的方法在外部秩序中不存在或者效果不好。国际秩序与其说是遵循国际法，不如说是丛林法则。

大国之间在军事实力相当、生存问题上存在不可调和矛盾的时候，最可能爆发战争。国家之间的斗争主要有5种类别：贸易/经济战、技术战、地缘政治战、资本战和军事战。

买枪炮（军事实力）需要财力，买黄油（国内社会支出需求）也需要财力。

> 个人启发：先进技术、战略资源→硬通货（芯片、石油、知识产权）→获取国际储备货币（黄金、美元）→提升军事实力、提升国民消费。

避免战争的方法：通过友好协商实现双赢。

本章重点阐述了第二次世界大战（二战）在国际秩序下爆发的原因，建议阅读原文。

### 7 从大周期的角度思考投资

所有市场都是由以下4个决定因素驱动的：经济增长、通胀、风险溢价和折现率，这4个决定因素的变化驱动投资回报的变化。

政府通过财政和货币政策影响这些因素。因此，政府目标与当前形势的相互作用成为周期的驱动因素。

金融财富（贷款）被创造出来之后，随着时间的推移会严重超出实物财富，最终，支付承诺远远超过支付能力，导致债务违约、经济萧条和股市崩盘等危机，进而形成了周期。

## 第二部分 500年世界发展史

### 8 过去500年简述

从1500年开始，财富和权力的分配方式，推动着社会的变革，一些重大的思维方式上的革命推动着巨大演变和进步：

1. 商业革命（12——16世纪）
    - 教会战争促进了海上贸易；
    - 一些意大利城邦更加灵活；
    - 威尼斯通过选举产生总督；
    - 金融体系建立信贷市场出现债券制度的存在与流行。
2. 文艺复兴（14——17世纪）
    - 逻辑推理超越神意成为主流解释；
    - 贸易、生产和银行业的进步；
    - 印刷机推动知识和思想快速传播；
3. 探险和殖民时代（15——18世纪）
    - 文艺复兴的技术成就转变为造船和航海技术的进步，远航船带回来的财富为文艺复兴的进一步发展提供了资金；
    - 西班牙和葡萄牙通过航海积累了大量的贵金属，并推高了物价；
    - 欧洲与中国的贸易，导致瓷器等商品贸易的发展，同时中国因不需要欧洲商品，导致大量的白银流入，也成为鸦片战争的最终成因；
    - 中国经历过郑和下西洋时代最辉煌的海军时刻后却选择了停止海上探索，而日本也在同时期选择了锁国；
4. 宗教改革（1517——1648年）
    - 宗教改革削弱了教会的统治地位，新教徒获得了大量的权利和自由；
5. 三十年战争（1648年）之后的新世界秩序
    - 《威斯特伐利亚合约》带来了各国地理边界的确立，并规定各国国民拥有决定本国事务的主权；
6. 资本主义的发明（17世纪）
    - 股票市场改善了资源分配，使得储蓄者将购买力转移给了企业家；
7. 科学革命（16——17世纪）
    - 科学方法的发展，加深了人类对世界的理解；
    - 科学技术的进步推动了经济增长；
8. 第一次工业革命（18——19世纪）
    - 用机器取代人力成为主流；
    - 城市化带来了新的社会结构；
    - 新技术的应用带来了大国之间的地位变化；
9. 启蒙运动与革命时代（17——18世纪）
    - 用科学方法来指导人的行为，带来君主权利衰落，个人的权利提升；
    - 带来了欧洲各国的革命浪潮；
10. 拿破仑战争及其后的新世界秩序（1803——1815年）
    - 拿破仑试图统一欧洲，但又最终失败，均势理论确定了各国新的边界和后续和平和繁荣，也开启了大不列颠治下的和平；
11. 西方列强进入亚洲（19世纪）
    - 英国等西方列强希望以武力迫使中国、日本、印度开放贸易，最终导致了中国清朝灭亡和洋务运动、日本内阁辞职和明治维新、印度被英国殖民；
12. 第二次工业革命（19世纪50年代至20世纪初）
    - 电力、电话、可互换的零部件及其他创新，进一步带来了贫富差距和资本市场的放纵行为；
13. 共产主义的出现（1848年）
    - 资本主义带来贫富差距，工业革命使新技术的拥有者获益，进而导致工人利益受损；
    - 资本家和工人阶级之间的冲突加剧，带来俄国和中国的共产主义变革；
