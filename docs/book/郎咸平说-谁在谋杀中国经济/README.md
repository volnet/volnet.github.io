## 前言 谋杀中国经济的文化魔咒

中国人底层的文化是投机取巧和浮躁的，投机取巧就是相信小概率事件，比如诸葛亮的各种小聪明，浮躁就是看事物只流于表面，相信失败是成功之母却不研究失败的理由。

中外文化差异也带来了中国人看世界和世界看中国人，互相看不懂，大家的思维模型是不一样的。

在文化差异的背景下重新看中国古代的商帮，很多原来对这些商帮的传统印象，可能就是错的，而正是这些传承下来的文化理念，对中国的现代企业也产生了很多根深蒂固的影响。
